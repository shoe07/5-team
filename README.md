# 골든타임 SOS - 설치 및 사용 가이드

## 📱 폰에서 앱으로 설치하는 방법

### Android (Chrome)
1. Chrome으로 앱 파일을 열거나 서버에서 접속
2. 우측 상단 **⋮ 메뉴** 클릭
3. **"홈 화면에 추가"** 탭
4. 이름 확인 후 **"추가"** 버튼 클릭
5. 홈 화면에 앱 아이콘 생성 완료 ✅

### iPhone (Safari)
1. Safari로 접속
2. 하단 **공유 버튼** (□↑) 클릭
3. **"홈 화면에 추가"** 선택
4. 이름 확인 후 **"추가"** 클릭

## 🚀 로컬에서 실행하기 (개발용)

```bash
# Python 3
python3 -m http.server 8080

# Node.js (npx)
npx serve .

# 이후 폰/브라우저에서
# http://[PC-IP]:8080 으로 접속
```

## ✨ 주요 기능

| 기능 | 설명 |
|------|------|
| 🆘 SOS 버튼 | 1.5초 길게 누르면 자동 119 연결 + 위치 전송 |
| 📍 GPS 위치 | 실시간 좌표 수신 및 구글맵 링크 생성 |
| ⚡ AED 목록 | 주변 자동제세동기 위치 표시 |
| 🩸 의료정보 | 혈액형, 지병, 알레르기 저장 |
| 👨‍👩‍👧 보호자 연락 | SOS 시 보호자 정보 포함 발송 |
| 📴 오프라인 지원 | 설치 후 인터넷 없어도 사용 가능 |
